# My Project

A pixel companion that stays with you while you work. Can cycle 
through random dialogues to help you keep going by pressing left mouse button. Draggable and can be placed where ever you desire.

# font file 
https://fonts.google.com/specimen/Pixelify+Sans (add font of your choice)
1. extract the zip file 
2. add it to project folder (\C:\Users\You\Downloads\ProjectFolder)
